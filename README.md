# Saavn-Downloader

Python Script for the MP3 downloader from Saavn.com

### Installation 

- Get a Token from [@BotFather](https://t.me/BotFather) and fill in `TG_BOT_TOKEN`.
- Run `pip install -r requirements.txt`.
- Run `python3 saavn-downloader.py`.
